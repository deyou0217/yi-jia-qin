# 🏠 一家亲 App

> 无服务器 · 纯P2P · 端对端加密 · 跨平台 · 绿色免安装

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🔐 端对端加密 | PBKDF2 派生密钥 + AES-256-GCM 全程加密 |
| 🌐 无服务器 | 信令仅用于建立连接，业务数据全走P2P |
| 💨 阅后即焚 | 拒绝文件后内存立即GC，不落磁盘 |
| 👨‍👩‍👧‍👦 最多10人 | Mesh拓扑，无需服务器转发 |
| 📱 跨平台 | Windows/Android/iOS/HarmonyOS/PC |
| 🟢 绿色免安装 | 单HTML文件，双击即用，删除即卸载 |

## 🚀 使用方法

### 最简方式（推荐）
直接双击打开 `一家亲App.html`，浏览器即可运行。

### 使用流程
1. **创建房间**：点击"创建房间"→ 系统自动生成房间号 → 设置密码和昵称
2. **分享房间号**：把房间号+密码告知家人（电话/微信告知即可）
3. **家人加入**：家人点击"加入房间"→ 输入相同房间号+密码 → 进入
4. **连接彼此**：进入聊天后，点击顶部 ➕ → 互相粘贴对方的节点ID建立连接
5. **开始使用**：文字聊天 / 视频通话 / 发送文件（对方可拒绝，拒绝则自动销毁）

## 🛡️ 隐私设计

```
用户密码
   ↓ PBKDF2 (100000轮)
AES-256-GCM密钥 (仅存内存)
   ↓
所有消息/文件 加密后 → WebRTC DataChannel → 对方设备 → 解密
   
没有账号注册、没有数据库、没有云端存储、没有日志
App关闭 = 所有数据消失
```

## ⚙️ 技术栈

- **前端框架**：Vue 3 (Composition API)
- **通信核心**：WebRTC + PeerJS DataChannel
- **加密算法**：Web Crypto API (PBKDF2 + AES-256-GCM)
- **信令服务**：PeerJS公共服务器（仅交换IP/端口，不存任何数据）
- **STUN服务器**：Google STUN + Twilio STUN（仅用于穿透NAT）
- **打包方案**：单HTML文件（零依赖，双击即用）

## 📝 注意事项

1. **局域网**：同一WiFi下连接速度最快，无需互联网
2. **外网**：需要两端都能访问公网（大多数家庭网络均可）
3. **NAT穿透**：极少数运营商级NAT可能无法直连（约5%情况）
4. **iOS视频**：iOS Safari对WebRTC支持较好，但需HTTPS或localhost环境
5. **文件大小**：建议单文件不超过100MB（受内存限制）

## 🔒 安全声明

- 密码**从不**离开用户设备
- 消息内容**从不**经过任何中间服务器
- 文件传输**从不**落地磁盘（除非用户明确同意保存）
- 关闭App后，所有会话数据**立即**从内存消失
